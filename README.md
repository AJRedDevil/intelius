# Synopsis

The objective of the project is to scrap the user phone number, email and address details of every search.

## Technologies

1. puppeteer
2. cheerio
3. csv-writer
4. csvtojson

## Installation

1. Install node
2. Install the npm packages.
```bash
npm install
```

## Run

1. Add email and password of intellius login in .env file in root path.
```bash
email=""
password=""
```
2. npm run start
